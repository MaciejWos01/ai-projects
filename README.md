# ai-projects
Showcase of AI focused projects
Interactive Dashboard: Data Visualization project showing Spotify analysis visualizations of songs and authors
Article Recommender: Information Retrieval project which recommends articles based on previous articles' similarity
Image Classifier: Deep Learning project focused on implementing an image classifier from scratch
Language Identifier: Natural Language Processing project using multinomial Naive Bayes classifier to predict the language of a given text
Self-driving Robot: Robotics project using CNN to train robot to follow the road
